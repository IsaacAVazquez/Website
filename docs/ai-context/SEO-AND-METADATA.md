## Metadata Generation

### constructMetadata() — Primary Builder
**File:** `src/lib/seo.ts`

```typescript
function constructMetadata(options: {
  title: string;
  description: string;
  canonicalUrl?: string;
  datePublished?: string;
  dateModified?: string;
  aiMetadata?: {
    profession?: string;
    expertise?: string[];
    topics?: string[];
    contentType?: string;
    context?: string;
    summary?: string;
    primaryFocus?: string;
    industry?: string[];
  };
}): Metadata
```

**Produces:**
- `title` with template: `Isaac Vazquez – [page-title]`
- `metadataBase` for relative URL resolution
- OpenGraph (og:title, og:description, og:image, og:type)
- Twitter card: `summary_large_image`
- Canonical URL
- Robots: index/follow, googleBot with unlimited preview
- AI-specific meta tags (when `aiMetadata` provided)

**Site config:**
```
name: "Isaac Vazquez"
url: "https://isaacavazquez.com"
ogImage: "/opengraph-image" (1200x630)
```

### generateMetadata() — Dynamic Pages
Used by `[slug]` pages. Fetches data at request time and returns `Metadata`.

**File:** `next-sitemap.config.js`

**Site URL:** `https://isaacavazquez.com`

### Change Frequencies
- Default: `weekly`
- About/Resume/Contact: `monthly`

`/api/*`, `/_next/*`, `/404`, `/admin/*`

## SEO Patterns to Follow

### Adding a New Page
1. Export `metadata` using `constructMetadata()` with `aiMetadata` object
2. Add `<StructuredData>` component with appropriate type
3. Include breadcrumbs if nested route
4. Set canonical URL
5. Add to sitemap with appropriate priority in `next-sitemap.config.js`

### Adding a Blog Post
1. Create `content/blog/[slug].md` with frontmatter (title, excerpt, publishedAt, tags, category)
2. Sitemap auto-discovers it
3. Article structured data auto-generated on `/writing/[slug]` page
4. Priority assigned based on slug keywords

### AI Optimization Checklist
- Include `aiMetadata` in `constructMetadata()` call
- Add relevant expertise, topics, and content type
- Include a concise `summary` field
- Use `context` for additional information AI systems should understand
