# Section Header Component

## Purpose
Visually distinct headers that introduce major sections of content on a page.

---

## Standard Section Header Template

```markdown
## [Section Title]

[Brief description or value proposition (1-2 sentences)]

---
```

## Styling Guidelines

- **H2:** Heading 2 (32px), Bold, Warm Gray 700
- **Description:** Body text (16px), Regular, Warm Gray 600
- **Spacing Above:** Space 3XL (64px)
- **Spacing Below:** Space L (24px)
- **Divider:** 1px line, Warm Gray 200 (optional)
- **Alignment:** Left (default) or Center (featured sections)

---

## Content Guidelines

**Title:**
- Clear and descriptive (3-6 words)
- Use sentence case or title case consistently
- Avoid generic titles like "Section 1"

**Description:**
- 1-2 sentences maximum
- Explain what's in this section
- Create interest and set context

**Optional Elements:**
- Category tags
- Date/update information
- Call-to-action link
- Icon or emoji for visual interest

---

## Hierarchy Levels

### Primary Section (H2)
```markdown
## Main Section Title
```
Used for major page divisions

### Secondary Section (H3)
```markdown
### Subsection Title
```
Used for subsections within primary sections

### Tertiary Section (H4)
```markdown
#### Minor Heading
```
Used for detailed breakdowns

---

## Section Spacing

**Between Sections:**
- Primary to Primary: Space 4XL (96px)
- Primary to Secondary: Space 2XL (48px)
- Secondary to Secondary: Space XL (32px)

**Within Sections:**
- Header to Content: Space L (24px)
- Content to Content: Space M (16px)

## Accessibility

- Use semantic heading tags (h2, h3, h4)
- Maintain logical heading order
- Ensure sufficient color contrast
- Avoid headings as pure decoration
- Provide descriptive, unique headings

---

## SEO Considerations

- Include relevant keywords naturally
- Use descriptive heading text
- Maintain proper heading hierarchy
- Keep headings concise (<60 characters)
- Make headings contextually meaningful
