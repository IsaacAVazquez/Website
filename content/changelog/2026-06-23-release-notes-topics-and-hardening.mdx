This was a site-operations release as much as a product release:

- [Release notes](/release-notes) joined the site as a shorter, month-grouped companion to this changelog.
- Writing topic pages launched so the archive can be browsed by recurring themes instead of only by date.
- Findability improved across the homepage, header, live tools, and search experience, with search becoming easier to reach from the primary shell.
- Analytics and SEO picked up GA4 event tracking, a reference page for analytics events, and sitemap priority and changefreq metadata by content type.
- Security hardening added a staged report-only CSP, stricter auth handling, safer JSON-LD output, and email rate limiting.
- Performance and accessibility work kept large sports snapshots out of client bundles, trimmed image runtime, fixed mobile-menu focus order, repaired the compare-modal focus trap, and made filter state clearer to assistive tech.

The docs also got a maintenance pass, including the Spring 2026 roadmap, a shorter `CLAUDE.md`, and a prompting best-practices guide.
